# 공개 안전성 검토

## 공개 기준

난임 시술 데이터는 의료 인접 민감 영역이므로 원본 데이터, 개인정보 가능 자료, 대회 비공개 자료는 공개 대상에서 제외합니다. 저장소는 데이터 처리, 검증 설계, 모델링 판단을 설명하는 포트폴리오입니다.

## 검토 포인트

- 공개 자료: README, 모델링 문서, 공개 가능한 노트북/코드
- 제외 자료: 원본 의료 인접 데이터, token, credential, 개인 식별 가능 자료

## 리뷰어 안내

평가 대상은 민감 데이터 경계를 인식한 tabular ML pipeline 구성입니다.
